init

this is my version of an init script for a new installation of a debian based operationg
it will install my most used apps, copy over my configs, and basically set up the OS for daily use

i am choosing debian because most packages are .deb
other packages can be installed in a vm if need be

apps to be installed include:
- vs code ✅
- git ✅
- gparted ✅
- podman ✅
- python ✅
- node ✅
- obsidian ✅

uninstall:
- gnome maps ✅
- gnome doc scanner ✅
- gnome tour ✅
- evolution ✅

configs:
- git
- gnome
- firefox
- vscode
